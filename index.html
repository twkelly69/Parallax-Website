視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，指網站的背景比前景移動得更慢。這能在訪客向下滾動頁面時呈現 3D 效果，增加景深並帶來更沉浸的瀏覽體驗。視差的原理來自視覺錯覺：因為人眼會把較近的物體看得更大，遠處的物體似乎移動得更慢。這種錯覺長期被應用在不同媒介的視差效果中，用以營造真實感。視差最早出現在傳統動畫中，早在迪士尼的《白雪公主與七個小矮人》以及《超級瑪利歐》這類的電子遊戲裡就已使用。隨著 CSS 與 HTML 的進步，視差效果逐漸演變成今天網頁設計領域熟悉的樣貌。

較早的瀏覽器視差滾動實作之一，由網頁開發者 Glutnix 在 2007 年的部落格文章分享，包含使用 JavaScript 與 CSS 2 的示例程式碼與展示，支援當時的 IE6 等瀏覽器。但直到 2011 年，隨著 HTML5 與 CSS3 的出現，這項技術才在網頁設計師間流行起來。支持者認為，它是擁抱網路流動性的簡單方式。許多人使用視差背景讓使用者更投入，並改善網站整體體驗。不過，普渡大學在 2013 年發表的研究指出：「……儘管視差滾動增強了使用者體驗的某些方面，但並不一定提升整體使用者體驗。」


為什麼要使用視差圖形？

    設計師嘗試視差圖形有兩個主要原因：

  • 有助於講述故事

    視差圖形非常適合用來講故事。好的設計就在於溝通——當我們設計網站時，希望與訪客交流，並講述關於產品、服務或品牌的動人故事。視差滾動能讓訪客沉浸在獨特的線上體驗中，甚至可以打造單頁網站，讓用戶無需跳轉就能讀完關於產品或品牌的完整故事。

  • 有助於提升用戶參與度

    視差圖形還能用來提高用戶參與度。降低跳出率是網頁設計需要關注的目標之一，跳出率衡量訪客僅瀏覽一頁便點擊返回或關閉分頁的比例。高跳出率會影響搜尋排名，因此任何能降低跳出率的舉措都值得嘗試。精心製作的視差頁面能提供更具動感的體驗，促使用戶在頁面停留更久。


如果想在網站加入視差滾動效果，請留意以下提示。

  • 選擇性使用

    將視差滾動用在特定元素上比到處使用更有力量。可以放在頁眉、標題或僅限首頁。你希望視差效果強化設計，而不是分散用戶注意力。

  • 壓縮圖片

    視差滾動會使用多種媒體檔案和 CSS 形狀來營造深度感。別讓過大的圖片拖慢載入速度、影響效能與體驗。

  • 運用色彩

    視差滾動不僅關乎前景與背景的圖像，也與色彩有關。你可以在背景使用去飽和的色調，在前景放上更鮮活的顏色，進一步增強景深。
