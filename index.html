视差滚动效果

作者：Vineet Kumar

什么是视差滚动？

视差滚动是一种网页设计技巧，指网站的背景比前景移动得更慢。这样在访客向下滚动页面时会产生3D效果，增加景深并带来更沉浸的浏览体验。视差基于光学错觉：因为人眼会把离我们较近的物体看得更大，远处的物体似乎移动得更慢。这种错觉长期被应用在不同媒介的视差效果中，营造真实感。它最早出现在传统动画中，早在迪士尼的《白雪公主与七个小矮人》以及《超级马力欧》这样的电子游戏里就已使用。随着 CSS 与 HTML 的进步，视差效果逐渐演变成今天网页设计领域熟悉的样子。

其中一个较早的浏览器视差滚动实现由网页开发者 Glutnix 于 2007 年在博客文章中分享，包含使用 JavaScript 和 CSS 2 的示例代码与展示，支持当时的 IE6 等浏览器。但直到 2011 年，随着 HTML5 与 CSS3 的出现，这项技术才在网页设计师间流行起来。支持者认为，它是拥抱网络流动性的简单方式。许多人使用视差背景让用户更投入，并改善网站整体体验。不过，普渡大学在 2013 年发表的研究指出：「……尽管视差滚动增强了用户体验的某些方面，但并不一定提升整体用户体验。」


为什么要使用视差图形？

    设计师尝试视差图形有两个主要原因：

  • 有助于讲述故事

    视差图形非常适合用来讲故事。好的设计就在于沟通——当我们设计网站时，希望与访客交流，并讲述关于产品、服务或品牌的动人故事。视差滚动能让访客沉浸在独特的线上体验中，甚至可以打造单页网站，让用户无需跳转就能读完关于产品或品牌的完整故事。

  • 有助于提升用户参与度

    视差图形还能用来提高用户参与度。降低跳出率是网页设计需要关注的目标之一，跳出率衡量访客仅浏览一页便点击返回或关闭标签页的比例。高跳出率会影响搜索排名，因此任何能降低跳出率的举措都值得尝试。精心制作的视差页面能提供更具动感的体验，促使用户在页面停留更久。


如果想在网站加入视差滚动效果，请留意以下提示。

  • 选择性使用

    将视差滚动用在特定元素上比到处使用更有力量。可以放在页眉、标题或仅限首页。你希望视差效果强化设计，而不是分散用户注意力。

  • 压缩图片

    视差滚动会使用多种媒体文件和 CSS 形状来营造深度感。别让过大的图片拖慢载入速度、影响性能与体验。

  • 运用色彩

    视差滚动不仅关乎前景与背景的图像，也与色彩有关。你可以在背景使用去饱和的色调，在前景放上更鲜活的颜色，进一步增强景深。
